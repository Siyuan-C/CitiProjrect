## 注意事项
* 不要改动库/框架代码
* 务必在每一次commit中说明改动内容
* 在修改其他人写的代码前务必与对方充分沟通
* 由于jQuery库的`load`函数在本地`file`协议下会因跨域问题而不能正常执行，在本地打开网页时务必设法用`http`或其它支持跨域请求的协议打开
